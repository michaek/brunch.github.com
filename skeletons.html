Skeletons

A Brunch skeleton is basically an application boilerplate that provides a good starting point for new applications.

Creating new application with any skeleton is pretty simple:

brunch new <path-or-URI> [optional-output-dir]
# Example from GitHub
brunch new gh:brunch/dead-simple
# Example from a full URL
brunch new https://github.com/paulmillr/brunch-with-chaplin

Brunch doesn’t encourage you to use any particular technology, but application skeletons can start you on your way with a specific pattern.

The complete skeletons list is on GitHub. If you create your own open-source skeleton, feel free to add it!

Selected interesting skeletons:

  • Dead-simple brunch — super minimal skeleton, useful if you want to port existing application or use your own stack.
  • Brunch with Chaplin: HTML5 Boilerplate, Chaplin framework over Backbone.js. Main languages are CoffeeScript, Stylus and Handlebars. Pretty useful for big apps. There are also SASS, LiveScript and Bootstrap + Stickit alternatives available.
  • Backbone-based:
    • Brunch with banana pancakes: tasty. Main languages are JavaScript, Sass, LESS & Handlebars. Includes Backbone.Mediator & Twitter Bootstrap.
    • Brunch with hipsters: Modernizr, Lodash, Backbone-Mediator, Font Awesome, Brunch Auto-Reload. Main languages are Coffee, Stylus and Handlebars.
    • Brunch with Marionette, Aura-brunch and create.js canvas skeletons
  • Angular-based:
    • Angular brunch seed: AngularJS, Twitter Bootstrap. Main languages are CoffeeScript, Jade and Less. There are also sass + H5BP and LiveScript versions.
    • Brunch on asteroids — version with Jade, CoffeeScript, D3.js
  • Ember-based:
    • Brunch with Ember Reloaded: Handlebars, H5BP, Swag; based on the official ember starter kit
    • Brunch with Ember: Bootstrap, Handlebars
    • Brunch with Hampsters: Coffee, Stylus, Embem.js, Swag, Font Awesome and modernizr
    • Tapas with Ember: CoffeeScript, Stylus, Handlebars, Ember Data, Ember Model.

  • And others:
    • Spine on Brunch: with Spine.js. Main languages are LiveScript, Less and Jade.
    • Cordova-brunch which works nice with Apache Cordova (aka phonegap)
